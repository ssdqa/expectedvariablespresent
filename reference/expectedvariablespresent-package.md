# expectedvariablespresent: Evaluate Distribution of Expected Study Variables

The Expected Variables Present module, part of the larger SQUBA
ecosystem, checks for the presence of expected variables and presents a
variety of distributions of these variables in the dataset. It allows
investigators to visualize multiple variables for comparison and to show
the co-occurrence of variables in the same patient.
